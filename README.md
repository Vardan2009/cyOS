# cyOS

cyOS is a simple UNIX-like 32-bit Operating System for the x86 CPU architecture.

## Current features

- VGA Driver
- PS/2 Keyboard Driver
- Serial IO
- Disk IO (ATA/IDE)
- Filesystem support (FAT12, FAT16, FAT32, and exFAT)
- Paging and Upper-half kernel
- Kernel Syscalls
- User tools and libc
- ELF executables and user processes
- ANSI output rendering
- User-mode shell

## Future features

- Multitasking/multithreading
- Graphical Interface
